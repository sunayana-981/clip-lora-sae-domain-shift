# Sparse Autoencoders for LoRA-Adapted CLIP Under Domain Shift

Standalone project website for the anonymized ICML Workshop 2026 submission
“Sparse Autoencoders for LoRA-Adapted CLIP Under Domain Shift.”

## Preview locally

The site has no build step. From this directory, run:

```bash
python3 -m http.server 8000
```

Then open <http://localhost:8000>.

## Repository structure

```text
.
├── index.html
└── assets
    ├── css/site.css
    ├── images/
    ├── js/site.js
    └── paper/sparse-autoencoders-lora-clip-domain-shift.pdf
```
